# 🎨 デザインお題ジェネレーター

Webデザインの実践練習用に、ランダムなお題を生成するツールです。

**→ [アプリを使う](https://your-username.github.io/your-repo-name/)**

---

## 概要

「何を練習すればいいか分からない」を解決するツールです。業種・ターゲット・画像の種類・内容・ゴール・制約をランダムに組み合わせてお題を生成します。毎回違う条件が出るので、ポートフォリオ作りや模擬案件の練習に使えます。

## 機能

- **難易度3段階**（初級 / 中級 / 上級）で生成されるお題の複雑さが変わります
- **1クリック生成**で業種・ターゲット・画像の種類・内容・ゴール・制約を一括生成
- **コピーボタン**でお題をテキストとしてクリップボードにコピー
- **リスト編集**機能で自分だけのお題項目を追加できます（追加内容はブラウザに保存されます）

## 使い方

1. 難易度（初級 / 中級 / 上級）を選ぶ
2. 「🎲 お題を生成する」をクリック
3. 表示されたお題でデザインを作る
4. 「📋 コピー」でお題をメモ帳などに保存

### リストのカスタマイズ

「⚙️ リストを編集する」から、業種・ターゲットなど各カテゴリに自分で項目を追加できます。追加した項目はブラウザのlocalStorageに保存されるため、ページをリロードしても消えません。

## 難易度の違い

| 難易度 | 制約の数 | 業種の傾向 | ターゲットの粒度 |
|--------|----------|------------|-----------------|
| 初級 | 1〜2個 | カフェ・美容院など身近な業種 | 年代・性別のみ |
| 中級 | 2〜3個 | 幅広いジャンル | 職業・ライフスタイルも含む |
| 上級 | 4〜5個 | ニッチ・B2Bも含む | 悩みや価値観まで含む細かいペルソナ |

## 技術構成

- HTML / CSS / JavaScript（1ファイル完結）
- 外部APIなし・バックエンドなし
- データ保存：localStorage（カスタム項目のみ）
- ホスティング：GitHub Pages

## ローカルで動かす

リポジトリをクローンして `index.html` をブラウザで開くだけで動作します。サーバーは不要です。

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
open index.html  # macOS
# または index.html をブラウザにドラッグ＆ドロップ
```

## GitHub Pages への公開

1. このリポジトリをフォーク or クローンして自分のGitHubにプッシュ
2. リポジトリの **Settings → Pages** を開く
3. Branch を `main`、フォルダを `/ (root)` に設定して **Save**
4. 数分後に `https://kirimura0617.github.io/Theme_generation/` で公開されます
